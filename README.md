# NobleScript
ANTLR4/LLVM based compiler

# Planned features
## Milestone 1
- 2 types of variables: int and double,
- read and write operations,
- basic arithmetic operations,
- error messages,
- arrays,
- strings.

## Milestone 2
- compound statements,
- functions,
- variable scopes,
- dynamic types (type coercion).
